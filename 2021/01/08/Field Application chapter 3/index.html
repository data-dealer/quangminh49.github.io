Field Application chapter 3 - Django REST framework part 1

January 8, 2021

Từ bài này Mình sẽ đi sâu hơn về phần kỹ thuật, bao gồm cả Software Architect và Coding, Phần Backend mình làm với Django REST framework nên kiến thức Python cơ bản là rất cần thiết cho các bạn nào muốn làm theo.

Tại sao mình lại chọn Django REST framework (DRF) ?

  • Đầu tiên, DRF dùng Python, ngôn ngữ đơn giản, hiệu quả mà mình đã có hơn 4 năm kinh nghiệm.
  • DRF cung cấp bộ khung hầu như đã gồm tất cả những gì cần để phát triển một REST API hoàn thiện.
  • Bộ docs của DRF rất đầy đủ, theo mình thấy là đầy đủ nhất trong các REST Framework
